## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [https://api.winnipegtransit.com](https://api.winnipegtransit.com) and [https://onesignal.com/](https://onesignal.com)

2. Clone the repo
   ```sh
   git clone https://github.com/jobanpreet27/winnipeg-transit-webapp-nextjs.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
